Книжный интернет-магазин

Учебный дипломный проект — полнофункциональный веб-сайт для продажи книг с расширенными возможностями ознакомления с контентом.

Функционал

Авторизация и регистрация с функцией «Запомнить меня» (сессии/cookies)
Личный кабинет пользователя
Каталог книг с обложками, названиями и описаниями
Биография авторов для каждой книги
Поиск по названию книги
Демо-версия** каждой книги (ознакомительный фрагмент)
Контактные данные (адрес, телефон, email)

Технологии

- PHP (серверная логика, сессии, работа с БД)
- MySQL (хранение данных)
- HTML / CSS (адаптивный интерфейс)
- JavaScript (интерактивность)

Структура БД (основные таблицы)

- `users` — пользователи
- `authors` — авторы и их биографии
- `books` — книги (название, описание, демо-текст, обложка)
- `contacts` — контактная информация магазина

Как запустить проект

1. Установите локальный сервер (OpenServer / XAMPP / MAMP)
2. Склонируйте репозиторий в папку `htdocs` или `www`
3. Импортируйте файл `database.sql` в phpMyAdmin
4. Настройте подключение к БД в `config.php`
5. Откройте в браузере: `http://localhost/ваша_папка`

Перспективы развития

- Корзина и оформление заказа
- Онлайн-оплата
- Отзывы и рейтинг книг
- Рекомендательная система («другие книги автора»)
